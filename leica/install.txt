
INSTALL
-------

a] Install LAS 410 full instalation.

b] Reboot.

c] Install Leica IEEE 1394 FireWire PCI/PCIe drivers
   from Firmware configuration utility.

d] Install "imaging devices" camera generic Leica DFC driver.

e] Run Camera diagnostic tool.

f] Run LAS.

   Hardware setup:

   Microscope - [MZ16FA] -> automotor + COM1 -> [test]
   Camera     - [DFC490] -> [test]

g] Done.

i] Edit HW config -> Hardware configuration utility -> "run as administrator".
